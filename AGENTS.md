# Agent Instructions

This project is a beginner-friendly static personal website.

Future Codex sessions should keep it:

- simple
- static
- framework-free
- easy for a beginner to edit
- usable by opening `index.html` directly in a browser

Use only plain HTML and CSS unless JR explicitly asks for something else.

Do not add:

- React
- Next.js
- Tailwind
- Bootstrap
- Vite
- npm tooling
- build steps
- package managers
- JavaScript unless explicitly requested

Keep the code readable, semantic, responsive, and accessible.

Never commit or expose private files such as `.env`, API keys, Google tokens, or Twilio credentials.

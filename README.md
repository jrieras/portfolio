# JR Personal Website

This is a simple personal portfolio website for JR.

It uses only:

- `index.html`
- `styles.css`

There is no React, Next.js, Tailwind, Bootstrap, Vite, npm, or framework. You can open the site directly in a browser.

## Preview The Website

Open `index.html` in your browser.

On a Mac, you can also run this from the project folder:

```bash
open index.html
```

## Edit The Website

Most content lives in `index.html`.

Common edits:

- Change the headline in the hero section.
- Update the selected work cards.
- Replace the writing links.
- Replace the contact placeholders.

Visual styling lives in `styles.css`.

Common style edits:

- Colors are at the top under `:root`.
- Spacing values are at the top under `:root`.
- Card, button, and layout styles are grouped by section.

## Placeholders To Replace

In `index.html`, replace:

- `your-email@example.com`
- `LinkedIn` link currently set to `#`
- `GitHub` link currently set to `#`
- Writing article links currently set to `#`

## Publish With GitHub Pages

After this project is in a GitHub repository:

1. Go to the repository on GitHub.
2. Click `Settings`.
3. Click `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Choose the `main` branch.
6. Choose the root folder `/`.
7. Click `Save`.

GitHub will give you a public website URL after it finishes publishing.

## Important

Do not upload private files like `.env` to GitHub. This website does not need secrets or API keys.
